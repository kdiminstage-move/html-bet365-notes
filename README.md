# html-bet365-notes

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面主要作为技术演示、工具页面或静态内容展示使用，不针对任何特定网站或域名。仓库结构简单，便于快速部署与访问。

## 目录说明

```
html-bet365-notes/
├── index.html          # 仓库主页，包含页面列表
├── pages/              # 存放各个独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/             # 公共资源文件（CSS、JS、图片等）
    ├── style.css
    └── script.js
```

- `index.html`：仓库入口页面，提供所有归档页面的导航链接。
- `pages/`：每个 HTML 文件代表一个独立的功能页面，彼此无依赖关系。
- `assets/`：存放被多个页面引用的公共资源，如样式文件、脚本或图像。

## 页面归档说明

- 所有页面均为静态 HTML，可直接在浏览器中打开运行。
- 页面内容不涉及任何商业推广、营销承诺或诱导点击行为。
- 每个页面均保持轻量、简洁，便于归档与长期维护。
- 如页面需要外部资源（如字体、库），会在页面头部注明来源。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来修正错误、优化页面或添加新页面。
- 新增页面请放入 `pages/` 目录，并在 `index.html` 中添加导航链接。
- 公共资源请统一放入 `assets/` 目录，避免重复。
- 本仓库由社区共同维护，请遵守开源社区的通用行为准则。
